# 51-lesson
Учебный проект. Чат-приложение, использующее API OpenRouter для взаимодействия с различными моделями искусственного интеллекта.

Репозиторий оригинального кода - https://github.com/neuro-fill/51-lesson

Внесенные изменения:

- Проверка API ключа через /auth/key
- Генерация и сохранение пин-кода
- Вход по пин при следующем входе
- Кнопка сброса ключа
- Ключ хранится в SQLite, не требует .env
